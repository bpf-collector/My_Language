  3. 内联式(行内样式) --> style=""
**********************选择器**********************
  1. id选择器
  2. 类选择器
  3. html选择器
  4. 通配符选择器
  5. 父子选择器
    • 后代选择器(所有子元素)： 空格
    • 子选择器(直接子元素)： >
    • 相邻兄弟选择器(其后第一个)： +
    • 一般兄弟选择器(其后所有)： ~
**********************属性选择器**********************
  • [属性] --> 选择具有指定属性的元素
  • [属性 = 值] --> 选择具有指定属性和值的元素
  • [属性 ~= 值] --> 选择含有指定值的属性的元素
  • [属性 |= 值(单词)] --> 选择具有指定值开头的属性的元素
  • [属性 ^= 值] --> 选择具有指定值开头的属性的元素
  • [属性 $= 值] --> 选择具有指定值结尾的属性的元素
  • [属性 *= 值] --> 选择具有包含指定值的属性的元素
**********************背景**********************
  • 背景颜色 --> background-color: 颜色
  • 背景图片 --> background-image: url("图片路径") z-index: (正|负)重叠
  • 背景图片重复 --> background-repeat: (repeat-x; repeat-y; no-repeat)
  • 背景图片滚动 --> background-attachment: (fixed; scroll)
  • 背景图片位置 --> background-position: (left; right) (top; center; bottom)
  • 速记属性 --> background: color image repeat attachment position
**********************边框**********************
  1. 类型 --> border-style:
    • dotted - 定义虚线边框
    • dashed - 定义虚线边框
    • solid - 定义实线边框
    • double - 定义双边框
    • groove - 定义3D凹槽边框。效果取决于边框颜色值
    • ridge - 定义3D脊状边界。效果取决于边框颜色值
    • inset - 定义3D嵌入边框。效果取决于边框颜色值
    • outset - 定义3D开始边框。效果取决于边框颜色值
    • none - 没有边界
    • hidden - 定义隐藏边框
  2. 宽度 --> border-width: (top; right; bottom; left)(px; pt; cm; em)
  3. 颜色 --> border-color: (top; right; bottom; left)颜色
  4. 速记属性 --> border: width style color
    • border-top
    • border-right
    • border-bottom
    • border-left
  5. 圆角边框 --> border-radius: px
**********************轮廓**********************
  • 类型 --> outline-style: 同border-style
  • 颜色 --> outline-color: 颜色
  • 宽度 --> outline-width: 宽度
  • 速记属性 --> outline: width style color
  • 偏移 --> outline-offset: px
**********************文字**********************
  • 文字颜色 --> color: 颜色 opacity： 透明度
  • 文字对齐 --> text-align: (left; right; center)
  • 文字装饰 --> text-decoration: (underline; none; ...)
  • 文字转换 --> text-transform: (uppercase; lowercase; capitalize)
  • 文字缩进 --> text-indent: px
  • 字母间距 --> letter-spacing: px
  • 单词间距 --> word-spacing: px
  • 行高 --> line-height: num
  • 文字方向 --> dircetion: (rtl; ltr; ...)
  • 文字阴影 --> text-shadow: (水平 垂直 颜色)
**********************字体**********************
  • 字体样式 --> font-style:
  • 字体大小 --> font-size: (px; em; %)
  • 字体重量 --> font-weight: (normal; bold; ...)
  • 字体变体 --> font-variant: (normal; small-caps; ...)
  • 响应字体 --> font-size: vw
**********************列表**********************
  • 标记样式 --> list-style-type: (circle; square; upper-roman; lower-alpha)
  • 列表图像 --> list-style-image: url("图片路径")
  • 标记位置 --> list-style-position: (inside; outside)
  • 速记属性 --> list-style: type position image
